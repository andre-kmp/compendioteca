# compendioteca

Literal presentation:

> In Portuguese, "compêndio" means "a compilation containing a summary of the essential points on a topic", and the suffix "-teca" means set, collection, cabinet, or storage place. "Compêndio" + "teca" is compendioteca, that is, the place where the summaries of the books I've been reading are going to be kept. That's the idea here.
